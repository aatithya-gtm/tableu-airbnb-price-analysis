# Airbnb Pricing & Geographic Trends Analysis (Tableau)

## 📌 Project Overview
Navigating short-term rental markets requires deep insight into geographic distributions and pricing dynamics. This project features a dynamic, interactive dashboard built using **Tableau Public** to analyze Airbnb listing data. The visualization provides real-time exploration of rental prices across different neighborhoods, room types, and listing densities, translating complex spatial data into actionable business intelligence.

## 🛠️ Data Visualization & Dashboard Features
This project utilizes intermediate and advanced Tableau features to maximize user experience and analytical depth:
* **Geographic Mapping:** Developed interactive map views plotted by location to visualize spatial price clusters and listing density.
* **Dynamic Filtering:** Implemented user controls for room types (e.g., Entire Home vs. Private Room), price ranges, and regional boundaries.
* **Key Performance Indicators (KPIs):** Integrated top-level metrics tracking total listing counts, average market prices, and inventory availability at a single glance.
* **Color Theory & Hierarchy:** Utilized a clean, professional dark/light contrast theme to make critical data points and pricing variances immediately stand out.
